Agent Skill : hf-jobs

在 Hugging Face Jobs 上运行任何工作负载。

当您想在 Hugging Face Jobs 上运行 GPU/CPU 工作负载（批量推理、合成数据生成、数据集统计、实验）时使用此技能，具有正确的令牌处理和结果持久化回 Hub 的功能。

技能名称： hf-jobs

主要文档： hf-jobs/SKILL.md

脚本目录： hf-jobs/scripts/

参考资料目录： hf-jobs/references/

概述

此技能专注于通过 Hugging Face Jobs 运行实际工作负载。它包括现成的 UV 脚本和指南，用于身份验证（HF 令牌）、secrets 与环境变量、超时、硬件选择以及将结果推送到 Hub。
